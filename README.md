# Desarrollo Frontend — Soluciones de ejercicios

Repositorio oficial con las soluciones a los ejercicios del libro [Desarrollo Frontend](https://leanpub.com/desarrollo-frontend) de [Óscar Bustos](http://oscarbustos.dev/) ([@elchiconube](https://twitter.com/elchiconube)).

## Estructura

```
soluciones/
├── html/
│   ├── ejercicio-01.md
│   ├── ejercicio-02.md
│   └── ...
├── css/
│   ├── ejercicio-01.md
│   ├── ejercicio-02.md
│   └── ...
└── javascript/
    ├── ejercicio-01.md
    ├── ejercicio-02.md
    └── ...
```

Cada archivo contiene el enunciado del ejercicio y su solución. Ten en cuenta que en muchos ejercicios no existe una única solución correcta — si la tuya funciona y el código es limpio y legible, es válida.

## Cómo usar este repositorio

Intenta resolver cada ejercicio por tu cuenta antes de consultar la solución. El aprendizaje más eficiente es el que se hace con las manos en la masa.

Si tienes dudas sobre algún ejercicio o encuentras algún error, puedes:

- Escribirme un [correo](mailto:oscarbustosgarrido@gmail.com)
- Contactarme a través de [X](https://twitter.com/elchiconube)

## Herramientas recomendadas

- Editor de código: [Visual Studio Code](https://code.visualstudio.com/)
- Navegador: [Google Chrome](https://www.google.com/chrome)
- Referencia HTML y CSS: [MDN Web Docs](https://developer.mozilla.org/)
